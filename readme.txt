== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Woo Tab</strong>" activate it.<br />

After activate plugin you will see "Woo Tab" menu at left side on WordPress dashboard click "New Woo Tab" and use the options field "Woo Tab Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When Tabs options setup done please publish Tabs as like post or page<br />

and then copy shortcode from top of <strong>Woo Tab Options</strong> `[woo_tab  id="1234" ]`<br />

product grid shortcode `[woo_tab_grid content_source="recent" total_items="10"]`

short-code args

* content_source -> recent, featured, on_sale, 
* total_items -> integer value


then paste this shortcode anywhere in your page to display Tabs<br />
